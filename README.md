# 混乱五子棋

### Preparatory stage:

Make sure you have install node and npm
node not lower than v20.11.0
npm not lower than 10.2.4

### Steps to run:

```
git clone https://github.com/luyaocode/chaos-gomoku.git
or
git clone https://gitee.com/luyaocoding/chaos-gomoku.git

npm install
npm run start
```
